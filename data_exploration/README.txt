****File / Folder Descriptions
FeedFiles -- where you put the files you want to load into python
ProcessedFiles -- where python will store your manipulated dataframes
data_analysis.py -- this it the python script you use to run your code

****To use this tool use the steps below

1) drop the file you want to use in the FeedFiles Folder
2) Open data_analysis in your favorite python scripting tool (spyder, atom, etc..)
3) Use the loading functions to load your data file (in FeedFiles folder) to python
4) manipulate the dataframe to your hearts content
5) use the csvFileSave function to save your dataframe as a CSV into the ProcessedFiles Folder
